Open source

At Zynesis, we believe in giving back to the open source community that has benefited not only us, but to advancement of technology. We have been consistently open sourcing our work and contributing to existing open source projects.

Opauth

Opauth is a multi-provider authentication framework for PHP that is authored initially by U-Zyn Chua.

Opauth enables PHP applications to perform user authentication on 3rd party providers, eg. Facebook Connect, Twitter Login, etc., really easily. Opauth is available as plugins for many PHP frameworks and CMSes (eg. CakePHP, Yii, Symfony, Laravel).

Try out Opauth or fork the open source project at GitHub.
